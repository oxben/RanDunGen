# RanDunGen

Random Dungeon Generator made with Godot Engine

## Requirements
* Godot Engine 3.2.x

## Usage
* **G**: Generate a new dungeon with a random seed
* **Q**: Quit

## Sources
* RanDunGen/        : The Godot project (project.godot, scenes and gdscripts)
* RanDunGen/assets/ : The assets (gfx, sfx, ...)
